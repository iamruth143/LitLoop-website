## 📖 About

**Lit Loop** is a book rental platform that makes reading accessible and affordable. Instead of buying books you'll read once, rent them at a fraction of the price and have them delivered straight to your door.

This repository contains the **official landing page** for Lit Loop, hosted at [litloop.in](https://litloop.in).

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🔍 **Smart Search** | Find books by title, author, genre, or series |
| 📦 **Doorstep Delivery** | Books delivered to your home or college |
| 💰 **Affordable Rentals** | Starting at just ₹6 per rental — no subscriptions |
| 🔔 **Real‑Time Tracking** | Live rental status updates and delivery notifications |
| ❤️ **Favorites & Lists** | Save books and build personal reading lists |
| 📍 **Location‑Based** | Google Maps–powered delivery zones |

## 🚀 Getting Started

### Prerequisites

Any static file server — no build tools required.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/iamruth143/LitLoop-website.git
cd LitLoop-website

# Serve with any static server (e.g. using npx)
npx serve .
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

---

## 🌐 Deployment

This is a fully static site. Deploy to any hosting provider:

- **GitHub Pages** — Push to `main` and enable Pages in repo settings
- **Netlify / Vercel** — Connect the repo for automatic deploys
- **Custom Domain** — Configured for [litloop.in](https://litloop.in)
